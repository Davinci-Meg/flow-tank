## 1. プロダクト概要

Flow Tankは「水がたまる」メタファーを用いたポモドーロタイマー Web アプリです。作業セッション中に抽象的な水槽に水が溜まっていくビジュアルで「集中している感覚」を可視化し、ユーザーの継続的な集中を促します。

- **ターゲット：** フリーランス・リモートワーカー・学生など、集中力を高めたい個人ユーザー
- **プラットフォーム：** Webブラウザ（デスクトップ・モバイルレスポンシブ）
- **デザインコンセプト：** 高級感のある北欧風・ライトモード

## 2. デザインシステム

### カラーパレット

| カラー名 | 値 | 用途 |
|----------|------|------|
| オフホワイト | `#FAF8F5` | 背景・ベース |
| ベージュ | `#F0EBE3` | カード・サーフェス |
| くすみブルー | `#7AACBF` | 水・アクセント・リンク |
| ディープネイビー | `#1B3A4B` | 見出し・強調テキスト |
| ウォームグレー | `#8C8279` | 副次テキスト・ラベル |
| ダークテキスト | `#2C2C2C` | 本文テキスト |

### タイポグラフィ

- **見出し：** Inter または Noto Sans JP（クリーンで可読性の高いサンセリフ）
- **本文：** 同フォントの Regular
- **数字・タイマー表示：** JetBrains Mono または DM Mono（等幅）

### 水槽ビジュアル

水槽は抽象的に表現します。ガラスやアクリル素材のリアルな描写ではなく、水位の上昇をグラデーション（くすみブルーの濃淡）と線（水面の揺らぎ）で表現します。魚や植物などの装飾要素は一切なし。CSS/SVG アニメーションを用い、空の状態から満水までスムーズに遷移します。

## 3. 機能仕様

### 3.1 ポモドーロタイマー

| 項目 | 詳細 |
|------|------|
| 作業時間 | デフォルト25分、ユーザーがカスタマイズ可能（5〜60分） |
| 休憩時間 | デフォルト5分（カスタム可）、長休憩デフォルト15分 |
| サイクル | 4セッションで長休憩（カスタム可） |
| 操作 | スタート / 一時停止 / リセット / スキップ |
| 水槽連動 | タイマー開始で注水開始、進捗率に応じて水位が上昇 |
| 完了演出 | 満水時に水面がキラキラ光るアニメーション + 穏やかな音 |
| 中断時 | 水が徐々に蒸発/減少するアニメーション |
| 休憩中 | 水面が穏やかに揺らぐアイドルアニメーション |

### 3.2 作業ラベル

セッション開始前または開始後に「作業内容」をテキストで入力可能。セッション完了時にラベルとともに記録され、統計やカレンダーでの内訳表示に使用されます。直近のラベル履歴からサジェスト表示も行います。

### 3.3 認証・アカウント

| 項目 | 詳細 |
|------|------|
| 未ログイン | タイマーのみ利用可。データはローカル保存。 |
| ログイン方法 | メール + パスワード / Google OAuth / GitHub OAuth |
| 会員限定機能 | ToDoリスト、カレンダー、統計ダッシュボード、タイマー設定のクラウド同期 |
| プロバイダー | Supabase Auth |

### 3.4 ToDoリスト（会員限定）

| フィールド | 型 | 備考 |
|-----------|-----|------|
| タイトル | テキスト | 必須 |
| 期限 | 日付 | 任意。過ぎたら赤色表示 |
| 優先度 | 高 / 中 / 低 | カラードットで視覚化 |
| 完了ステータス | チェックボックス | 完了済みはグレーアウト |

ポモドーロセッション開始時に ToDo からタスクを選んで作業ラベルとして自動リンクすることも可能。

### 3.5 カレンダー（会員限定）

月表示のカレンダー。各日にその日のセッション完了数と合計集中時間を表示します。セッション数に応じて水滴アイコンの濃さが変わるヒートマップ風のデザインで、過去の努力が一目でわかります。日付をクリックするとその日のセッション詳細（作業ラベル・時間）がポップアップで表示されます。

### 3.6 統計ダッシュボード（会員限定）

カレンダーとは別画面で、週間・月間の集中時間をグラフで可視化します。

| グラフ | 内容 |
|--------|------|
| 週間棒グラフ | 曜日ごとの集中時間（分） |
| 月間折れ線グラフ | 日ごとの集中時間の推移 |
| サマリーカード | 今週の合計・今月の合計・連続日数・平均セッション数 |
| ラベル内訳 | 作業ラベル別の時間配分（ドーナツチャート） |

## 4. 画面構成

| 画面 | パス | アクセス |
|------|------|---------|
| ホーム（タイマー） | `/` | 全ユーザー |
| ログイン / 新規登録 | `/auth` | 未認証 |
| ToDoリスト | `/todos` | 会員のみ |
| カレンダー | `/calendar` | 会員のみ |
| 統計ダッシュボード | `/stats` | 会員のみ |
| 設定 | `/settings` | 会員のみ |

ナビゲーションはサイドバー（デスクトップ）またはボトムタブ（モバイル）で切り替え。未認証ユーザーには会員限定機能のタブに「ログインして解放」のバッジを表示します。

## 7. 主要 UX フロー

### 初回訪問（未ログイン）

1. ホームにアクセス → 空の水槽とタイマーが表示
2. 作業ラベルを入力（任意）→ スタートを押す
3. 水が徐々にたまる → 25分後に満水エフェクト
4. 「ログインすると記録が保存されます」のさりげない CTA を表示

### 会員ユーザーの日常利用

1. ログイン済みでホームにアクセス
2. ToDo からタスクを選択、またはラベルを手入力
3. セッション完了 → 自動保存 → カレンダーに反映
4. 統計ページで週間・月間の振り返り

## 8. 非機能要件

| 項目 | 要件 |
|------|------|
| レスポンシブ | モバイルファースト。375px〜1440pxで快適に動作 |
| パフォーマンス | タイマーアニメーション 60fps。水槽は CSS/SVG で軽量に |
| オフライン対応 | タイマーはオフラインでも動作。オンライン復帰時に同期 |
| アクセシビリティ | キーボードショートカット対応（Spaceでスタート/ストップ） |
| セキュリティ | Supabase RLSでユーザー単位のデータ分離 |
